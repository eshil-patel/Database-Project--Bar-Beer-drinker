Modifications

Modifications MUST be done via the SQL Query Interface. Use the information below to do so.

This page will provide you with query formats to use on the SQL Query Interface.
These query formats will allow you to Insert, Delete, or Update any row in our database.

For INSERT, you should only be editting the values within VALUES(...). See example for details.

For UPDATE you can edit any of the values in a row. Take the Update example below and add or remove any fields you want to update.

If the query is succesful, the SQL Query Interface will print that out. E.g. "Succesful Insert".
If there is an error with syntax or your statement triggers our Event Condition Actions (ECAs), an error will print out.

Example for Bars:
INSERT INTO bars (`barId`, `barName`, `license`, `open`, `close`, `phone`, `address`, `city`, `state`) VALUES (1000, 'exName', 'exLicense', '12:00', '13:00', '123456789', '10 Example St.', 'Trenton', 'NJ');

DELETE FROM bars WHERE barId = 10;

UPDATE bars SET `barName` = 'exampleName', `license` = 'exampleLicense', WHERE `barId` = 0;
*You can add or remove column names if you wish to change how many columns you modify!*

The statements above can be conducted on any of the tables below as long as they conform to the Relationship Schema.
Use this schema to insert, modify, or delete from any of our tables:
